# Store Assets

These PNG files are prepared as Chrome Web Store listing starters:

- `promo-small-440x280.png`: small promotional image.
- `screenshot-new-tab-1280x800.png`: new tab page screenshot-style asset.
- `screenshot-add-panel-1280x800.png`: custom search engine panel screenshot-style asset.

Before final submission, compare them with a real unpacked-extension screenshot in Chrome or Edge and replace them if the UI has changed.
